## 已驗證通過功能（Passed）

### 帳號登入
- 使用正確帳號密碼可成功登入
- 未輸入帳號或密碼時顯示正確提示
- 輸入錯誤帳號密碼時登入失敗並顯示提示
- 忘記密碼與創建帳號入口連結正確

### 忘記密碼
- 進入忘記密碼頁面流程正確
- 未填 Email 或格式錯誤時顯示正確提示
- 輸入正確 Email 後送出重設通知

### 後台帳號管理
- 管理員可停用帳號並即時更新狀態
- 被停用帳號嘗試登入時系統正確拒絕
- 管理員後台可修改使用者資料；一般使用者介面不顯示此操作

### 測驗題庫完整性
- 48 個月（4歲）個案可正常開始測驗，無「題目不足」錯誤
- 60 個月（5歲）個案可正常開始測驗，無「題目不足」錯誤
- 72 個月（6歲）個案可正常開始測驗，無「題目不足」錯誤

### 測驗邊界穩定性
- 最低年齡層（2個月）個案持續答錯時，系統維持穩定不崩潰

## 尚未驗證功能（Not Evaluated）

以下功能因技術限制或環境限制尚未完成自動化驗證，**不代表功能有問題**：

| 功能 | 原因 | 建議行動 |
|------|------|---------|
| 帳號創建（register） | SIT 環境帳號創建功能未開放 | 確認是否預計開放，或改在 UAT 測試 |
| 帳號註冊驗證碼（account-register） | 需模擬 Email 驗證碼，測試工具尚未設定 | 工程師設定驗證碼 mock |
| 圖卡配對（card-matching） | 需取得測驗頁面詳細截圖後確認 selector | 工程師補充截圖後補測 |
| 步態分析（gait-analysis） | 同上，需步態模組頁面截圖 | 工程師補充截圖後補測 |
| 觀察題組（observation-group） | 需等待 AI 影片分析完成（外部系統） | 考慮以固定已完成個案進行手動測試 |
| 口語表達（verbal-expression） | 需模擬計時器計時（60 秒），技術調整中 | SDET 調整後補測 |
| 影片錄製（video-recording） | 需模擬相機錄製環境 | 建議手動測試確認 |
| 重新錄製（re-recording） | 需等待影片上傳完成 | 建議手動測試確認 |
| 手繪圖形辨識（handwriting-recognition） | 部分案例需人工視覺確認 | 安排 QA 人工測試 |
| 資料驗證（data-validation） | 身分證驗證需 API 測試工具（pytest），尚未設定 | SDET 設定後補測 |
| 題目邏輯（升降階，question-logic） | 需模擬答對/答錯的 API 回應，技術調整中 | SDET 調整後補測 |
